Verwendung von position:absolute

Auf dieser Beispielseite möchten wir die Positionierung mit CSS zeigen. In der rechten oberen Ecke des main-Elements wird eine rot umrahmte, absolut positionierte Box angezeigt. Damit die absolute Positionierung auf die rechte obere Ecke des main-Elements Bezug nimmt, musste dieses mit position:relative als Bezugsrahmen definiert werden.
