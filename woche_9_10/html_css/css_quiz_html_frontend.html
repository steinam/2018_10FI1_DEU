Aufbau der HTML-Datei¶

Die Optik des Frontends lässt auf folgende Strukturen im HTML schließen:

Die HTMl-Datei besteht aus den folgenden Abschnitten

  • HEAD:

    Hier werden die Meta-Informationen definiert sowie das Stylesheet eingebunden

  • BODY:

    Der Body ist eine Zusammensetzung von verschachtelten DIV-Elementen. Er besteht aus dem Kopf- und Fußbereich und dem dazwischen liegenden Inhalt. Die 10 Fragen werden ebenfalls mit DIVs modelliert und später per CSS als Inline-Block definiert. Die Antworten sind individuelle Buttons. Der darin enthaltene Text wird später per Javascript eingefügt. Die Fortschritts- und Zeitanzeige im Fußbereich besteht aus p-Elementen und wird ebenfalls per Javascript gesteuert.

    Im Ergebnis sieht die reine HTML-Datei dann wie folgt aus. Die Optik ist ziemlich unspektakulär.
